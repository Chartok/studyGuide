Notes on HTML

  • Element refers to the opening and closing tag.
  • The tag refers to the text inside the angle brackets
  • Not all elements have a closing tag and are known as self-closing tags
  • A closing tag begins with a /
  • The browser interperets the content between the opening and closing tags based on the tag name
  • HTML attributes defined in the beginning of the opening tag of an element provides additional information about the element
  • Elements nested in other elements are called child elements.
  • The element that holds child elements is known as the parent element.
  • The header appears at the top and usually contains navigation menu and perhaps even introductory content.
  • The footer appears at the bottom of the page and sometimes contains the author, copyright, contact, and more.
  • Not all elements are semantic; div creates a division in the webpage, span creates an inline division.
  • The !DOCTYPE html declares to the browser the document type is html.
  • Best practice to use semantic elements when possible, especially when visible to users.

  • The head element contains metadata that web browsers use to bring the webpage to the user.
  • The meta element defines the metadata such as: character set, description, keywords, and more.
  • The attribute charset="UTF-8" sets the character set to UTF-8 that allows the use of most characters and symbols.
  • The lang="en" attribute inside the html opening tag tells the browser the language for the webpage.
  • The http-equiv element sets an HTTP header for content.
  • The viewport attribute sets the default view for our webpage.
  • The title element defines and sets the title of the webpage.
  • The style element defines the CSS styles associated with a website.
  • The link element connects and external resource to the HTML document.
  • The base element defines the base URL for a webpage.
  • The script element embeds client-side scripts in the HTML document.
  • The body element represents the visible content shown to the user.
  • The h1-h6 elements refer to the heading level of the associated text block. The h1 heading element should only be used once on each webpage
  • The p element represents a paragraph or block of text, and is used to wrap most of the text on webpages.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items. Used for bulleted or numbered lists
  • The img element contains information about images that are displayed on the webpage. It can contain various attributes, src attribute defines the location and source of the image, and the alt attribute describes the appearance and functionality of the image and used for accessability
  • The br element creates a line of empty space, or a line break between two content blocks.
  • The a element creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link
  • The id attribute allows HTML to identify and select specific elements.
  • The class attribute works in a similar way as the id attribute.
