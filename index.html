HTML

  • Element refers to the opening and closing tag.
  • The tag refers to the text inside the angle brackets
  • Not all elements have a closing tag and are known as self-closing tags
  • A closing tag begins with a /
  • The !DOCTYPE html declares to the browser the document type is html.
  • Best practice to use semantic elements when possible, especially when visible to users.
  • The head element contains metadata that web browsers use to bring the webpage to the user.
  • The meta element defines the metadata such as: character set, description, keywords, and more.
  • The attribute charset="UTF-8" sets the character set to UTF-8 that allows the use of most characters and symbols.
  • The lang="en" attribute inside the html opening tag tells the browser the language for the webpage.

CSS

  • most common way to style a webpage is by using an external CSS style sheet
  • link file using link element in HTML's head element
  • selector defines the element or attributes to which the declarations apply
  • Declarations in CSS are totally different than declarations in HTML
    • Declarations consist of two parts:
      • CSS property
      • Property's value
    • Example of CSS rule:
      .card {
      text-align: center;
      font-size: 30px;
      }
  • The period preceding a selector designates this as a class selector. Any HTML element assigned to the class, will gain these property assignments from the CSS.
  • box-shadow is a CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas.
    • described by X and Y offsets relative to the element
    • blur and spread radius (not defined here, more later)
    • color
  • Assigning colors in CSS can be done in many ways, including by semantic references
    • can also refer to a color by its hexadecimal code
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
