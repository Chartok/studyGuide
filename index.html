HTML


  • Element refers to the opening and closing tag.
  • The tag refers to the text inside the angle brackets
  • Not all elements have a closing tag and are known as self-closing tags
  • A closing tag begins with a /
  • The !DOCTYPE html declares to the browser the document type is html.
  • Best practice to use semantic elements when possible, especially when visible to users.

  1. The head element contains metadata that web browsers use to bring the webpage to the user.
  2. The meta element defines the metadata such as: character set, description, keywords, and more.
  3. The attribute charset="UTF-8" sets the character set to UTF-8 that allows the use of most characters and symbols.
  4. The lang="en" attribute inside the html opening tag tells the browser the language for the webpage.
